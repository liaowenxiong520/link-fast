# 获取代理实例列表接口

## 基本信息

| 项目           | 说明                                            |
|--------------|-----------------------------------------------|
| 接口地址         | `http://112.124.51.99/api/instance/list`      |
| 请求方式         | `GET`                                         |
| Content-Type | `application/x-www-form-urlencoded`（Query 参数） |
| 返回格式         | `application/json`                            |

## 请求参数

### 必传参数

| 参数名       | 类型      | 必填 | 校验规则        | 说明                                             |
|-----------|---------|:--:|-------------|------------------------------------------------|
| proxyType | Integer | ✅  | 不能为空        | 代理类型                                           |
| status    | Integer | ✅  | 不能为空        | 实例状态（1=待创建, 2=创建中, 3=运行中, 6=已停止, 10=关闭, 11=释放） |
| pageNum   | Integer | ✅  | ≥ 1         | 页码                                             |
| pageSize  | Integer | ✅  | ≥ 1 且 ≤ 100 | 每页条数                                           |

### 可选参数

| 参数名         | 类型      | 必填 | 说明           |
|-------------|---------|:--:|--------------|
| countryCode | String  | ❌  | 国家代码         |
| cityCode    | String  | ❌  | 城市代码         |
| ip          | String  | ❌  | IP地址（支持模糊查询） |

### 请求示例

```
GET /api/instance/list?proxyType=101&status=3&pageNum=1&pageSize=10&countryCode=US&ip=192.168
```

{
    "code": 200,
    "message": "success",
    "data": {
        "total": 100,
        "totalPages": 10,
        "pageNum": 1,
        "pageSize": 10,
        "list": [
            {
                "ip": "192.168.1.100",
                "port": 8080,
                "regionId": "us-east-1",
                "countryCode": "US",
                "cityCode": "NYC",
                "status": 3,
                "username": "proxy_user_001",
                "pwd": "a1b2c3d4",
                "instanceNo": "c_gz3h2igp6dz9cq3",
                "orderNo": "ORD20260315001",
                "productNo": "PRD_101_US",
                "userExpired": 1750000000,
                "createTime": "2026-03-15 10:30:00"

### 外层结构（Result）

| 字段      | 类型      | 说明           |
|---------|---------|--------------|
| code    | Integer | 状态码，200 表示成功 |
| message | String  | 响应消息         |
| data    | Object  | 分页数据对象       |

### 分页结构（PageResult）

| 字段         | 类型      | 说明     |
|------------|---------|--------|
| total      | Integer | 总记录数   |
| totalPages | Integer | 总页数    |
| pageNum    | Integer | 当前页码   |
| pageSize   | Integer | 每页条数   |
| list       | Array   | 代理实例列表 |

### 实例字段（ProxyInstanceVO）

| 字段          | 类型      | 说明                                             |
|-------------|---------|------------------------------------------------|
| ip          | String  | 代理地址                                           |
| port        | Integer | 代理端口                                           |
| regionId    | String  | 区域地址                                           |
| countryCode | String  | 国家代码                                           |
| cityCode    | String  | 城市代码                                           |
| status      | Integer | 实例状态（1=待创建, 2=创建中, 3=运行中, 6=已停止, 10=关闭, 11=释放） |
| username    | String  | 代理账户名                                          |
| pwd         | String  | 代理密码                                           |
| instanceNo  | String  | 平台实例编号                                         |
| orderNo     | String  | 平台订单号                                          |
| productNo   | String  | 产品编号                                           |
| userExpired | Long    | 到期时间（Unix 时间戳，单位：秒）                            |
| createTime  | String  | 创建时间                                           |

## 错误响应示例

### 参数校验失败

```json
{
    "code": 400,
    "message": "参数校验失败: proxyType: 代理类型proxyType不能为空",
    "data": null
}
```

### 服务器异常

```json
{
    "code": 500,
    "message": "系统异常，请稍后重试",
    "data": null
}
```
