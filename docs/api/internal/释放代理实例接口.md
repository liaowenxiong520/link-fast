# 释放代理实例接口

## 基本信息

| 项目           | 说明                   |
|--------------|----------------------|
| 接口地址         | `/api/order/release` |
| 请求方式         | `POST`               |
| Content-Type | `application/json`   |
| 返回格式         | `application/json`   |

## 请求参数

请求体为 **JSON 对象**。

### 请求体字段

| 参数名         | 类型       | 必填 | 说明                   |
|-------------|----------|:--:|----------------------|
| payPassword | String   | ✅  | 支付密码（6位数字）           |
| instanceNos | String[] | ✅  | 平台实例编号列表，至少传入一个      |

### 请求示例

```json
{
    "payPassword": "168888",
    "instanceNos": [
        "c_gjhfpkxyqjx2n76",
        "c_gjhfpkxyqjx2n77"
    ]
}
```

> 支持批量释放，`instanceNos` 中可传入多个实例编号。

## 返回数据

### 通用响应结构

| 字段      | 类型      | 说明                          |
|---------|---------|:-----------------------------|
| code    | Integer | 状态码：200=成功，400=参数错误，500=系统错误 |
| message | String  | 提示信息                        |
| data    | Object  | 业务数据                        |

### 成功响应

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "appOrderNo": "F20329587392622171151741888888000",
        "orderNo": "C20260328103256675599",
        "status": 1,
        "amount": 0.00
    }
}
```

### data 字段说明（ProxyReleaseResultVO）

| 字段         | 类型         | 说明                      |
|------------|------------|:-------------------------|
| appOrderNo | String     | 渠道商释放订单号（前缀为 `F`）        |
| orderNo    | String     | 平台订单号                   |
| status     | Integer    | 订单状态：1=待处理              |
| amount     | BigDecimal | 本次释放退款金额（单位：元，无退款时为 0） |

## 错误响应示例

### 支付密码错误（code: 400）

```json
{
    "code": 400,
    "message": "支付密码错误",
    "data": null
}
```

### 参数校验失败（code: 400）

```json
{
    "code": 400,
    "message": "参数校验失败: payPassword: 支付密码不能为空",
    "data": null
}
```

### 释放失败-第三方返回业务错误（code: 500）

```json
{
    "code": 500,
    "message": "释放失败: 实例不存在或已释放",
    "data": null
}
```

### 释放失败-系统异常（code: 500）

```json
{
    "code": 500,
    "message": "释放代理失败，请稍后重试",
    "data": null
}
```
