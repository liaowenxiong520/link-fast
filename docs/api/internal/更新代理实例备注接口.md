# 更新代理实例备注接口

## 基本信息

| 项目           | 说明                       |
|--------------|---------------------------|
| 接口地址         | `/api/instance/remark`    |
| 请求方式         | `PUT`                     |
| Content-Type | `application/json`        |
| 返回格式         | `application/json`        |

## 请求参数

### 请求体字段

| 参数名        | 类型     | 必填 | 说明                        |
|------------|--------|:--:|---------------------------|
| instanceNo | String | ✅  | 平台实例编号                    |
| remark     | String | ❌  | 备注内容，不传或传空字符串均表示清空该实例的备注  |

### 请求示例

```json
{
    "instanceNo": "c_gjhfpkxyqjx2n76",
    "remark": "这是我的美国节点"
}
```

清空备注示例：

```json
{
    "instanceNo": "c_gjhfpkxyqjx2n76",
    "remark": ""
}
```

## 返回数据

### 成功响应

```json
{
    "code": 200,
    "message": "success",
    "data": null
}
```

### 通用响应结构

| 字段      | 类型      | 说明                           |
|---------|---------|:-----------------------------|
| code    | Integer | 状态码：200=成功，400=参数错误，500=系统错误 |
| message | String  | 提示信息                         |
| data    | Object  | 固定为 null                     |

## 错误响应示例

### 参数校验失败-instanceNo 为空（code: 400）

```json
{
    "code": 400,
    "message": "参数校验失败: instanceNo: 实例编号instanceNo不能为空",
    "data": null
}
```

### 实例不存在（code: 400）

```json
{
    "code": 400,
    "message": "实例不存在: c_gjhfpkxyqjx2n76",
    "data": null
}
```
