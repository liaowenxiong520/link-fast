# 获取代理实例列表接口

## 基本信息

| 项目           | 说明                                            |
|--------------|-----------------------------------------------|
| 接口地址         | `http://112.124.51.99/api/instance/list`      |
| 请求方式         | `GET`                                         |
| Content-Type | `application/x-www-form-urlencoded`（Query 参数） |
| 返回格式         | `application/json`                            |

## 请求参数

### 必传参数

| 参数名      | 类型      | 必填 | 校验规则        | 说明   |
|----------|---------|:--:|-------------|------|
| pageNum  | Integer | ✅  | ≥ 1         | 页码   |
| pageSize | Integer | ✅  | ≥ 1 且 ≤ 100 | 每页条数 |

### 可选参数

| 参数名         | 类型        | 必填 | 说明                                                    |
|-------------|-----------|:--:|-------------------------------------------------------|
| proxyType   | Integer[] | ❌  | 代理类型，支持传多个值，不传则不限制                                    |
| status      | Integer   | ❌  | 实例状态（1=待创建, 2=创建中, 3=运行中, 6=已停止, 10=关闭, 11=释放），不传则不限制 |
| countryCode | String    | ❌  | 国家代码                                                  |
| cityCode    | String    | ❌  | 城市代码                                                  |
| ip          | String    | ❌  | IP地址（支持模糊查询）                                          |

### 请求示例

```
GET /api/instance/list?pageNum=1&pageSize=10
GET /api/instance/list?proxyType=101&proxyType=103&status=3&pageNum=1&pageSize=10&countryCode=US&ip=192.168
```

    "code": 200,

### 外层结构（Result）

| 字段      | 类型      | 说明           |
|---------|---------|:-------------|
| code    | Integer | 状态码，200 表示成功 |
| message | String  | 响应消息         |
| data    | Object  | 分页数据对象       |

### 分页结构（PageResult）

| 字段         | 类型      | 说明     |
|------------|---------|:-------|
| total      | Integer | 总记录数   |
| totalPages | Integer | 总页数    |
| pageNum    | Integer | 当前页码   |
| pageSize   | Integer | 每页条数   |
| list       | Array   | 代理实例列表 |

### 实例字段（ProxyInstanceVO）

| 字段          | 类型      | 说明                                             |
|-------------|---------|:-----------------------------------------------|
| ip          | String  | 代理地址                                           |
| port        | Integer | 代理端口                                           |
| regionId    | String  | 区域ID                                           |
| regionName  | String  | 区域中文名，格式为「国家-城市」，如「美国-纽约」                      |
| status      | Integer | 实例状态（1=待创建, 2=创建中, 3=运行中, 6=已停止, 10=关闭, 11=释放） |
| username    | String  | 代理账户名                                          |
| pwd         | String  | 代理密码                                           |
| instanceNo  | String  | 平台实例编号                                         |
| renew       | Integer | 是否自动续费（0=否, 1=是）                               |
| orderNo     | String  | 平台订单号                                          |
| productNo   | String  | 产品编号                                           |
| unit        | Integer | 购买时的周期单位（1=天, 3=月, 4=年）                        |
| duration    | Integer | 购买时的周期时长（如 duration=30、unit=1 表示30天）           |
| userExpired | Long    | 到期时间（Unix 时间戳，单位：秒）                            |
| remark      | String  | 备注                                             |
| createTime  | String  | 创建时间                                           |

## 错误响应示例

### 参数校验失败

```json
{
    "code": 400,
    "message": "参数校验失败: proxyType: 代理类型proxyType不能为空",
    "data": null
}
```

### 服务器异常

```json
{
    "code": 500,
    "message": "系统异常，请稍后重试",
    "data": null
}
```
