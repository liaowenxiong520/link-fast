# 获取代理产品列表接口

## 基本信息

| 项目           | 说明                                            |
|--------------|-----------------------------------------------|
| 接口地址         | `http://112.124.51.99/api/proxy-product/list` |
| 请求方式         | `GET`                                         |
| Content-Type | `application/x-www-form-urlencoded`（Query 参数） |
| 返回格式         | `application/json`                            |

## 请求参数

### 必传参数

| 参数名         | 类型      | 必填 | 校验规则        | 说明   |
|-------------|---------|:--:|-------------|------|
| countryCode | String  | ✅  | 不能为空        | 国家代码 |
| cityCode    | String  | ✅  | 不能为空        | 城市代码 |
| pageNum     | Integer | ✅  | ≥ 1         | 页码   |
| pageSize    | Integer | ✅  | ≥ 1 且 ≤ 100 | 每页条数 |

### 可选参数

| 参数名       | 类型         | 必填 | 说明                  |
|-----------|------------|:--:|---------------------|
| proxyType | Integer 数组 | ❌  | 代理类型列表，为空或不传时查询全部类型 |

### 请求示例

```
GET /api/proxy-product/list?countryCode=US&cityCode=NYC&pageNum=1&pageSize=10&proxyType=101&proxyType=102
```

## 返回数据

### 数据结构

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "total": 50,
        "totalPages": 5,
        "pageNum": 1,
        "pageSize": 10,
        "list": [
            {
                "productNo": "PRD_101_US_NYC",
                "productName": "美国纽约静态住宅代理",
                "proxyType": 101,
                "protocol": "socks5",
                "detail": "美国纽约静态住宅IP，独享带宽",
                "unit": 2,
                "duration": 30,
                "costPrice": 5.50,
                "inventory": 200
            }
        ]
    }
}
```

### 外层结构（Result）

| 字段      | 类型      | 说明           |
|---------|---------|--------------|
| code    | Integer | 状态码，200 表示成功 |
| message | String  | 响应消息         |
| data    | Object  | 分页数据对象       |

### 分页结构（PageResult）

| 字段         | 类型      | 说明     |
|------------|---------|--------|
| total      | Integer | 总记录数   |
| totalPages | Integer | 总页数    |
| pageNum    | Integer | 当前页码   |
| pageSize   | Integer | 每页条数   |
| list       | Array   | 代理产品列表 |

### 产品字段（ProxyProductVO）

| 字段          | 类型         | 说明                  |
|-------------|------------|---------------------|
| productNo   | String     | 产品编号                |
| productName | String     | 产品名称                |
| proxyType   | Integer    | 代理类型                |
| protocol    | String     | 协议类型（HTTP/SOCKS5）   |
| detail      | String     | 详情描述                |
| unit        | Integer    | 单位                  |
| duration    | Integer    | 最小时长                |
| costPrice   | BigDecimal | 成本价                 |
| inventory   | Integer    | 库存数量                |

## 错误响应示例

### 参数校验失败

```json
{
    "code": 400,
    "message": "参数校验失败: countryCode: 国家代码countryCode不能为空",
    "data": null
}
```

### 服务器异常

```json
{
    "code": 500,
    "message": "系统异常，请稍后重试",
    "data": null
}
```
