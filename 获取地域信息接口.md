# 获取地域信息接口

## 基本信息

| 项目           | 说明                                            |
|--------------|-----------------------------------------------|
| 接口地址         | `/api/area/tree`                              |
| 请求方式         | `GET`                                         |
| Content-Type | `application/x-www-form-urlencoded`（Query 参数） |
| 返回格式         | `application/json`                            |

## 请求参数

| 参数名   | 类型           | 必填 | 说明                        |
|-------|--------------|:--:|---------------------------|
| codes | String 数组    | ❌  | 地域代码列表，不传则返回全部地域树 |

### 请求示例

```
GET /api/area/tree                          -- 获取全部地域树
GET /api/area/tree?codes=US&codes=CN        -- 获取指定地域
```

## 返回数据

### 数据结构

返回的是四级树形结构：**洲 → 国家 → 州/省 → 城市**

```json
{
    "code": 200,
    "message": "success",
    "data": [
        {
            "code": "1",
            "name": "Asia",
            "cname": "亚洲",
            "children": [
                {
                    "code": "CHN",
                    "name": "China",
                    "cname": "中国",
                    "children": [
                        {
                            "code": "CHN013",
                            "name": "Hebei",
                            "cname": "河北",
                            "children": [
                                {
                                    "code": "CHN013001",
                                    "name": "Shijiazhuang",
                                    "cname": "石家庄",
                                    "children": null
                                },
                                {
                                    "code": "CHN013002",
                                    "name": "Tangshan",
                                    "cname": "唐山",
                                    "children": null
                                }
                            ]
                        }
                    ]
                }
            ]
        }
    ]
}
```

### 外层结构（Result）

| 字段      | 类型      | 说明           |
|---------|---------|--------------|
| code    | Integer | 状态码，200 表示成功 |
| message | String  | 响应消息         |
| data    | Array   | 地域树形列表       |

### 地域节点（AreaVO，递归嵌套）

| 字段       | 类型            | 说明                 |
|----------|---------------|--------------------|
| code     | String        | 地域代码               |
| name     | String        | 地域英文名称             |
| cname    | String        | 地域中文名称             |
| children | Array\<AreaVO\> | 下级地域列表，末级节点为 null |

### 树形层级说明

| 层级 | 含义   | code 示例        | cname 示例 |
|:--:|------|----------------|----------|
| 1  | 洲    | `1`            | 亚洲       |
| 2  | 国家   | `CHN`          | 中国       |
| 3  | 州/省  | `CHN013`       | 河北       |
| 4  | 城市   | `CHN013001`    | 石家庄      |

## 错误响应示例

```json
{
    "code": 500,
    "message": "获取地域列表失败，请稍后重试",
    "data": null
}
```
