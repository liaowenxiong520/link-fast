# 获取代理订单列表接口

## 基本信息

| 项目           | 说明                                    |
|--------------|---------------------------------------|
| 接口地址         | `/api/order/list`                     |
| 请求方式         | `GET`                                 |
| Content-Type | 无（Query 参数）                           |
| 返回格式         | `application/json`                    |

## 请求参数

### Query 参数

| 参数名       | 类型      | 必填 | 说明                                        |
|-----------|---------|:--:|-------------------------------------------|
| status    | Integer | ✅  | 订单状态：1=待处理, 2=处理中, 3=处理成功, 4=处理失败, 5=部分完成 |
| pageNum   | Integer | ✅  | 页码，从1开始                                   |
| pageSize  | Integer | ✅  | 每页条数                                      |
| orderType | String  | ❌  | 订单类型：1=新建, 2=续费, 3=释放                     |
| orderNo   | String  | ❌  | 订单号（模糊查询）                                 |

### 请求示例

```
GET /api/order/list?status=3&pageNum=1&pageSize=10
```

```
GET /api/order/list?status=1&pageNum=1&pageSize=10&orderType=1&orderNo=P2026
```

## 返回数据

### 通用响应结构

| 字段      | 类型      | 说明                           |
|---------|---------|------------------------------|
| code    | Integer | 状态码：200=成功，400=参数错误，500=系统错误 |
| message | String  | 提示信息                         |
| data    | Object  | 分页数据对象                       |

### 成功响应

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "total": 25,
        "totalPages": 3,
        "pageNum": 1,
        "pageSize": 10,
        "list": [
            {
                "orderNo": "P202603151200001",
                "orderType": "1",
                "amount": 11.00,
                "instanceTotal": 2,
                "userId": 10001,
                "createTime": "2026-03-15 12:00:00"
            },
            {
                "orderNo": "P202603151200002",
                "orderType": "1",
                "amount": 25.50,
                "instanceTotal": 5,
                "userId": 10001,
                "createTime": "2026-03-15 14:30:00"
            }
        ]
    }
}
```

### data 字段说明（PageResult）

| 字段         | 类型               | 说明        |
|------------|------------------|-----------|
| total      | Integer          | 总记录数      |
| totalPages | Integer          | 总页数       |
| pageNum    | Integer          | 当前页码      |
| pageSize   | Integer          | 每页显示条数    |
| list       | Array\<Object\>  | 当前页的数据列表  |

### list 数组元素字段说明（ProxyOrderVO）

| 字段            | 类型         | 说明                           |
|---------------|------------|------------------------------|
| orderNo       | String     | 平台订单号                        |
| orderType     | String     | 订单类型：1=新建, 2=续费, 3=释放        |
| amount        | BigDecimal | 订单总金额                        |
| instanceTotal | Integer    | 订单对应实例总数量                    |
| userId        | Long       | 买家用户ID                       |
| createTime    | String     | 创建时间（格式：yyyy-MM-dd HH:mm:ss） |

## 错误响应示例

### 参数校验失败（code: 400）

缺少必传参数时：

```json
{
    "code": 400,
    "message": "参数校验失败: status: 订单状态status不能为空",
    "data": null
}
```

### 系统异常（code: 500）

```json
{
    "code": 500,
    "message": "系统繁忙，请稍后再试",
    "data": null
}
```
