# 验证支付密码接口

## 基本信息

| 项目           | 说明                 |
|--------------|--------------------|
| 接口地址         | `/api/pay/verify`  |
| 请求方式         | `POST`             |
| Content-Type | `application/json` |
| 返回格式         | `application/json` |

## 请求参数

### 请求体结构

| 参数名         | 类型     | 必填 | 校验规则 | 说明         |
|-------------|--------|:--:|------|------------|
| payPassword | String | ✅  | 不能为空 | 支付密码（6位数字） |

### 请求示例

```json
{
    "payPassword": "168888"
}
```

## 返回数据

### 密码正确

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "passed": true,
        "message": "支付密码正确"
    }
}
```

### 密码错误

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "passed": false,
        "message": "支付密码错误"
    }
}
```

### 返回字段说明（PayPasswordVO）

| 字段      | 类型      | 说明                   |
|---------|---------|----------------------|
| passed  | Boolean | 校验是否通过：true=正确，false=错误 |
| message | String  | 提示信息                 |

## 错误响应示例

### 参数校验失败

```json
{
    "code": 400,
    "message": "参数校验失败: payPassword: 支付密码不能为空",
    "data": null
}
```

### 服务器异常

```json
{
    "code": 500,
    "message": "系统繁忙，请稍后再试",
    "data": null
}
```
