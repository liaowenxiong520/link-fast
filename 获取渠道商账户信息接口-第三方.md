# 获取App信息

## 请求路径
`/api/open/app/info/v2`

## 请求参数
无

## 返回数据

| 参数          | 二级参数 | 类型     | 必填 | 说明            |
|-------------|------|--------|----|---------------|
| appName     | -    | string | 是  | app名称         |
| coin        | -    | string | 是  | 账户余额          |
| credit      | -    | string | 是  | 授信额度          |
| useBridge   | -    | int    | 否  | 使用桥 1 不使用 2使用 |
| callbackUrl | -    | string | 否  | 回调地址          |
| status      | -    | int    | 否  | 1正常 -1禁用      |
