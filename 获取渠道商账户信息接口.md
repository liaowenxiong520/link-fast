# 获取渠道商账户信息接口文档

## 接口地址

`GET /api/account/info`

## 请求方式

GET

## 请求参数

无业务参数，直接请求即可。

## 请求示例

```
GET /api/account/info
```

## 响应参数

| 字段      | 类型      | 说明     |
|---------|---------|--------|
| code    | int     | 响应码    |
| message | string  | 响应信息   |
| data    | object  | 账户信息对象 |

### data（AccountInfoVO）结构

| 字段      | 类型      | 说明        |
|---------|---------|-----------|
| coin    | string  | 账户余额      |
| credit  | string  | 授信额度      |

## 响应示例

```
{
  "code": 200,
  "message": "success",
  "data": {
    "coin": 100.00,
    "credit": 1000.00,
  }
}
```

## 错误响应示例

```
{
    "code": 500,
    "message": "系统异常，请稍后重试",
    "data": null
}
```

## 说明

- 本接口用于获取渠道商账户信息，前端无需传递业务参数。
- 响应数据结构以实际返回为准。
