# 创建代理订单接口

## 基本信息

| 项目           | 说明                                    |
|--------------|---------------------------------------|
| 接口地址         | `/api/order/open`                     |
| 请求方式         | `POST`                                |
| Content-Type | `application/json`                    |
| 返回格式         | `application/json`                    |

## 请求参数

### 请求体结构

| 参数名           | 类型              | 必填 | 说明              |
|---------------|-----------------|:--:|-----------------|
| payPassword   | String          | ✅  | 支付密码（6位数字）      |
| orderType     | Integer         | ✅  | 订单类型：1=新建, 2=续费 |
| totalQuantity | Integer         | ✅  | 购买总数量           |
| params        | Array\<Object\> | ✅  | 购买代理产品列表        |

### params 数组中的对象字段

| 参数名         | 类型      | 必填 | 说明                                          |
|-------------|---------|:--:|---------------------------------------------|
| productNo   | String  | ✅  | 产品编号                                        |
| proxyType   | Integer | ❌  | 代理类型                                        |
| countryCode | String  | ❌  | 国家代码                                        |
| stateCode   | String  | ❌  | 州/省代码                                       |
| cityCode    | String  | ❌  | 城市代码                                        |
| unit        | Integer | ❌  | 单位：1=天, 2=周, 3=月, 4=年, 10=无限制               |
| duration    | Integer | ❌  | 产品时长                                        |
| count       | Integer | ❌  | 购买数量（实例个数），默认1                              |
| cycleTimes  | Integer | ❌  | 购买时长周期数，默认1                                 |

### 请求示例

```json
{
    "payPassword": "168888",
    "orderType": 1,
    "totalQuantity": 2,
    "params": [
        {
            "productNo": "ipideash_598",
            "proxyType": 101,
            "countryCode": "US",
            "stateCode": "",
            "cityCode": "LAX",
            "unit": 3,
            "duration": 1,
            "count": 2,
            "cycleTimes": 1
        }
    ]
}
```

## 返回数据

### 通用响应结构

| 字段      | 类型      | 说明                          |
|---------|---------|-------------------------------|
| code    | Integer | 状态码：200=成功，400=参数错误，500=系统错误 |
| message | String  | 提示信息                        |
| data    | Object  | 业务数据                        |

### 成功响应

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "appOrderNo": "203295873926221711517418888880000",
        "status": 1,
        "orderNo": "P202603151200001",
        "amount": 11.00
    }
}
```

### data 字段说明（ProxyOrderCreateVO）

| 字段         | 类型         | 说明                                   |
|------------|------------|----------------------------------------|
| appOrderNo | String     | 渠道商订单号                               |
| status     | Integer    | 订单状态：1=待处理                           |
| orderNo    | String     | 平台订单号（第三方调用失败时为 null）               |
| amount     | BigDecimal | 订单总金额（第三方调用失败时为 null）               |

## 错误响应示例

### 支付密码错误（code: 400）

```json
{
    "code": 400,
    "message": "支付密码错误",
    "data": null
}
```

### 参数校验失败（code: 400）

```json
{
    "code": 400,
    "message": "参数校验失败: payPassword: 支付密码不能为空",
    "data": null
}
```

### 订单创建失败（code: 500）

```json
{
    "code": 500,
    "message": "订单创建失败，请稍后重试",
    "data": null
}
```
